# Student Management System

A simple console-based Java application to manage student records.

## Features
- Add student
- View all students
- Search student by ID
- Update student by ID
- Delete student by ID
- Input validation for age

## Tech Used
- Java
- OOP Concepts
- ArrayList
- Git & GitHub
